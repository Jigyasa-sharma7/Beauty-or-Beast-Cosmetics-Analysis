# Beauty or Beast: Analyzing Harmful Chemicals in Cosmetics

## Overview

Beauty or Beast is a machine learning and data analytics project that analyzes cosmetic products based on their ingredients and predicts whether a product is likely to be discontinued. The project includes data preprocessing, exploratory data analysis (EDA), interactive visualizations, and a Streamlit-based web application for prediction and analysis.

## Features

* Interactive Streamlit dashboard
* Product search functionality
* Chemical analysis of cosmetic products
* Data visualizations (histograms, bar charts, heatmaps, pair plots, word cloud, etc.)
* Product discontinuation prediction using Random Forest
* Model evaluation using Confusion Matrix

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* WordCloud

## Project Files

* `app.py` – Streamlit application
* `Beauty_or_Beast_Analysis.ipynb` – Data preprocessing, EDA, and model development
* `cleaning.py` – Data cleaning script
* `final_cleaned_data.csv` – Processed dataset

## How to Run

1. Clone this repository.
2. Install the required libraries.
3. Run the application using:

```bash
streamlit run frontend.py
```
